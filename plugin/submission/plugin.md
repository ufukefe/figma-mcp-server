## Title
Figma MCP Server

## Tagline
Allow AI Agents to edit your designs!

## Description
Model Context Protocol(MCP) is the standard for communication for AI agents with external tools. The Figma MCP Server plugin gives Large Language Models(LLMs) access to your Figma document. Your copilot can now help you create a design, make the required changes, organize your design document, and read from it. The Figma MCP server could be used together with all popular AI copilots: ChatGPT, Claude Desktop, Cursor, Windsurf, Perplexity, etc.

You are limited only by your imagination and the LLM that you use!
